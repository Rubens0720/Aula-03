Propriedades Position

Existem cinco valores de posições diferentes

  • static
  • relative
  • fixed
  • absolute
  • stikey

Os elementos são posicionados usando as propriedades top, bottom, left e rigth

Position 'static'

os elementos em html por padrão já são posicionados em static. Nessa posição , eles nãosão afetados pelas propriedades, left, rigth e bottom

Exemplo

Position 'relative'

São afetados pelas propriedades top, left, rigth e bottom. O conteúdo não se ajusta para se encaixar no espaço deixado pelo elemento

Exemplo

Position Fixed

O elemento é posicionado em relação a viewport, ou seja, ele vai permanecer no mesmo lugar da página

Exemplo

Position absolute

O elemento é posicionado com relação ao seu ancestral mais próximo

Exemplo

Position Sticky

Um elemento sticky é posicionando com base na posição de rolagem do usúario. Ele alterna entre relative e fixed, dependendo da posição de rolagem.
